# Commande-BDK

Application web de gestion des commandes pour un laboratoire de production pâtisserie/boulangerie.

## Stack

- Next.js 16 + React 19
- Supabase (base de données + auth)
- Tailwind CSS 4
- TypeScript

## Lancer le projet

```bash
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).
